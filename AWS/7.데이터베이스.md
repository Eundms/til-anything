# Amazon Aurora : RDB
- 불필요한 I/O 작업을 줄일 수 있도록 설계된 관리형 관계형 데이터베이스
- MySQL, PostgreSQL과 호환

# Amazon DynamoDB : NoSQL
- `문서 데이터 구조`와 `키-값 데이터 구조` 양쪽 모두에 빠르고 예측 가능한 성능을 제공
- 프로비저닝된 용량으로 오토 스케일링

# Amazon ElastiCache : 메모리 캐싱 서비스 
- 인 메모리 캐시 시스템 관리의 복잡성을 줄일 수 있도록 구축
- Redis, Valkey 또는 Memcached


# Amazonn DocumentDB : MongoDB 호환

# Amazon Neptune : 그래프 데이터베이스 
- 소셜 네트워킹 애플리케이션에서 사용되는 데이터세트 같은 고도로 연결된 데이터세트를 관리하는 특별히 제작된 완전관리형 그래프 데이터베이스 서비스


# AWS BackUp 
- 백업 모니터링 및 관리를 위한 단일 대시보드 제공 
- Amazon Elastic Block Store(Amazon EBS) 볼륨, Amazon Elastic File System(Amazon EFS) 파일 시스템 및 다양한 데이터베이스를 비롯한 여러 스토리지 유형을 지원
